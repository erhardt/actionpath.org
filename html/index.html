Action Path

Action Path uses mobile technology and crowdsourcing to help Jane Jacobs-style monitorial citizens achieve greater impact.
Imagine you're in Kendall Square near MIT and Action Path sends you a site-specific notification.
You are asked to answer one-question survey about a proposed development project.
Using Action Path, you quickly learn about the issue, provide feedback, and subscribe to future updates about it.
