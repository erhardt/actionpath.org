Action Path

Imagine you're in walking just outside of New Haven, CT and Action Path sends you a site-specific notification.
You are asked to follow-up on a report of illegal dumping in a nearby park.
Using Action Path, you quickly learn about the issue, update its status, and subscribe to monitor future updates about it.

Description

Action Path is a mobile app to help citizens provide meaningful feedback about everyday spaces in their cities by inviting engagement when you are near current issues.
